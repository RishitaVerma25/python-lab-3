# Library Inventory Manager

A Python project to manage library books using:
- Object-Oriented Programming
- JSON file handling
- CLI (Command Line Interface)
- Exception handling and logging

## Features
- Add books
- Issue books
- Return books
- Search books
- View all books
- Data stored in JSON

## How to Run the Program
Open terminal and run:
python cli/main.py

## Folder Structure
library_manager/ → Contains classes  
cli/ → Contains main program  
catalog.json → Stores book data
